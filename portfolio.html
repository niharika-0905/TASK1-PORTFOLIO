Sentiment Analysis for a Text given by the User

Sentiment analysis is like having a virtual buddy who can read between the lines of text to
figure out if someone is happy, sad, or frustrated. It’s all about using tech to understand
the emotions behind the words we type. In more detail, sentiment analysis uses natural 
language processing to analyze the words and phrases in a message to determine the overall 
sentiment conveyed. It looks for patterns and context clues to understand the emotions 
behind the text. By recognizing keywords, tone, and context, the system can categorize the 
sentiment as positive, negative, or neutral. It’s like having a friend who’s great at 
picking up on the vibes in a conversation
